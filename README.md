# Chasm Scout

## What is Scout

In the Chasm decentralised AI ecosystem, a scout is essentially a node that connected to an orchestrator to run LLM jobs.

Please see the [Setup](./docs/SETUP.md) to run the scout yourself

Supported Platform:

- Groq
- OpenRouter
- ollama
- OpenAI ChatGPT
